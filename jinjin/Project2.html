스트레스 지수(HRV) 값에 따른 상태

정상 범위 : 0 ~ 0.4
주의 : 0.4 ~ 0.7
높음 : 0.7 ~ 1.0

HRV값 계산 방법

1. 정규화(Normalization)
Normalized Heart Rate = Heart Rate - HR(min)/HR(max)-HR(min)

Normalized CO2 Level = CO2Level - CO2(min)/CO2(max)-CO2(min)

2. 스트레스 지수 계산
Stress Index = a X Normalized Heart Rate + b X Normalized CO2 Level
